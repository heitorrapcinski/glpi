# GLPI Fork

This repository is a fork of [GLPI](https://github.com/glpi-project/glpi), an open-source IT Asset Management and Service Desk software developed by [Teclib'](https://www.teclib-edition.com/) and the GLPI Project community.

## About the Original Project

GLPI (Gestionnaire Libre de Parc Informatique) is a free IT Asset Management, issue tracking system and service desk solution. It is licensed under the [GNU General Public License v3.0 or later](https://www.gnu.org/licenses/gpl-3.0.html).

- Website: https://www.glpi-project.org/
